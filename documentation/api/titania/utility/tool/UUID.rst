UUID
====

.. js:class:: Titania.Utility.Tool.UUID( )

   :returns: Une chaine de caractère.

   Génère un identifiant respectant les conventions de la RFC 4122 (d'après les spécifications de la version 4 des identifieurs).

   Cette chaine peut être considérée comme unique lors de l'exécution d'un programme. Si d'aventure ce n'était pas le cas ... eh bien espérez que personne ne s'en rende compte.
