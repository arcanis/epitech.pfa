Singleton
=========

.. js:class:: Titania.Utility.Tool.Singleton( definition )

   :param object definition: Définition de la classe à créer.
   :returns: Une instance de la classe créée.

   Fonctionne de la même façon que :js:class:`Titania.Utility.Tool.Class`, à une différence près : le retour de la fonction est une instance de la classe, et non la classe elle-même.
